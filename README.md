# PCA and LDA (2D Visualization)

This project demonstrates dimensionality reduction using:
- Principal Component Analysis (PCA)
- Linear Discriminant Analysis (LDA)

Both are applied with 2 components on the Iris dataset.

## Files
- pca_lda.ipynb : Main notebook
- dataset.csv : Dataset used
- result.png : 2D visualization result

## Libraries
pandas, matplotlib, scikit-learn
